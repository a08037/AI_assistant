# Нейропомощник AI

Веб-приложение на React + FastAPI для общения с AI-моделью через OpenAI API.

Приложение позволяет создавать несколько чатов, сохранять историю диалогов в рамках пользовательской сессии и получать ответы от AI в режиме реального времени (Streaming).

---

## Основные возможности

### Работа с чатами

* Создание новых чатов
* Переименование чатов
* Удаление чатов
* Автоматическое создание первого чата при запуске приложения
* Сброс пользовательской сессии

### Работа с AI

* Генерация ответов через OpenAI API
* Потоковая выдача ответа (Streaming)
* Поддержка системных промптов
* Выбор модели AI
* Сохранение истории переписки

### Готовые сценарии

* Планирование дня
* Создание чек-листов
* Генерация бизнес-идей
* Подготовка к собеседованиям
* Создание шаблонов документов
* Помощь в обучении

## Требования

Перед запуском должны быть установлены:

* Python 3.11+
* Node.js 18+
* npm
* OpenAI API Key

---

## Настройка OpenAI API

Создайте файл:

```text
backend/.env
```

Содержимое:

```env
OPENAI_API_KEY=ваш_api_ключ
SESSION_SECRET=my-secret-key
```

Установите пакет:

```bash
pip install python-dotenv
```

И загрузите переменные окружения в приложении.

---

## Запуск проекта

### 1. Запуск Backend

Перейдите в папку backend:

```bash
cd backend
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите сервер:

```bash
uvicorn app:app --reload
```

Backend будет доступен по адресу:

```text
http://localhost:8000
```

Документация API:

```text
http://localhost:8000/docs
```

---

### 2. Запуск Frontend

Откройте новый терминал:

```bash
cd frontend
```

Установите зависимости:

```bash
npm install
```

Запустите приложение:

```bash
npm run dev
```

Frontend будет доступен по адресу:

```text
http://localhost:5173
```

---

## Сборка проекта

Сборка frontend:

```bash
cd frontend
npm run build
```

Результат будет создан в папке:

```text
dist/
```

## Использование

1. Откройте приложение в браузере.
2. Создайте новый чат или используйте автоматически созданный.
3. Выберите модель AI.
4. При необходимости измените системный промпт.
5. Введите запрос.
6. Нажмите кнопку «Отправить».
7. Получите ответ от AI.

## Возможные проблемы

### Ошибка 400 Bad Request

Причина:

```text
OPENAI_API_KEY is not set
```

Решение:

Проверьте наличие переменной:

```env
OPENAI_API_KEY=...
```

---

### Не работает кнопка «Отправить»

Проверьте:

* создан ли чат;
* запущен ли backend;
* доступен ли OpenAI API ключ.
